# 🐳 Containerized Flask Web App

## 📌 About The Project

This repository contains a simple, lightweight Flask web application that is fully containerized using Docker. As an Information Technology engineering student focusing on full-stack development and cloud infrastructure, I built this project to demonstrate the seamless integration of web frameworks with modern DevOps tools. 

By containerizing the application, it ensures that the app runs identically across different environments, eliminating the classic "it works on my machine" problem. This project serves as a foundational step toward building more complex, cloud-ready microservices.

### 🛠️ Built With
* **Backend:** Python, Flask
* **Containerization:** Docker
* **OS Environment:** Linux (Alpine/Ubuntu)
